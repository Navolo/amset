Installation¶

Requirements¶

  • Python 3.6+
  • NumPy
  • SciPy
  • BoltzTraP2

Install¶

$ # Download the repository and install
$ git clone https://github.com/hackingmaterials/amset.git
$ cd amset
$ pip install -e .

Run tests¶

$ python setup.py test
