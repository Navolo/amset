class amset.interpolation.interpolate.Interpolater(band_structure, num_electrons, interpolation_factor=20, soc=False, magmom=None, mommat=None, interpolate_projections=False)¶

Class to interpolate band structures based on BoltzTraP2.

Details of the interpolation method are available in:

  1. Madsen, G. K. & Singh, D. J. Computer Physics Communications 175, 67–71 (2006)
  1. Madsen, G. K., Carrete, J., and Verstraete, M. J. Computer Physics
    Communications 231, 140–145 (2018)
Parameters:
  • band_structure (BandStructure) – A pymatgen band structure object.
  • num_electrons (int) – The number of electrons in the system.
  • interpolation_factor (float) – Factor used to determine the accuracy of the band structure interpolation. Also controls the k-point mesh density for Interpolater.get_amset_data().
  • soc (bool) – Whether the system was calculated using spin–orbit coupling.
  • magmom (Optional[ndarray]) – The magnetic moments for each atom.
  • mommat (Optional[ndarray]) – The band structure derivatives.
  • interpolate_projections (bool) – Whether to interpolate the band structure projections.
get_amset_data(energy_cutoff=None, scissor=None, bandgap=None, symprec=0.01, nworkers=-1)¶

Gets an AmsetData object using the interpolated bands.

Note, the interpolation mesh is determined using by interpolate_factor option in the Inteprolater constructor.

This method is much faster than the get_energies function but doesn’t provide as much flexibility.

The degree of parallelization is controlled by the nworkers option.

Parameters:
  • energy_cutoff (Optional[float]) – The energy cut-off to determine which bands are included in the interpolation. If the energy of a band falls within the cut-off at any k-point it will be included. For metals the range is defined as the Fermi level ± energy_cutoff. For gapped materials, the energy range is from the VBM - energy_cutoff to the CBM + energy_cutoff.
  • scissor (Optional[float]) – The amount by which the band gap is scissored. Cannot be used in conjunction with the bandgap option. Has no effect for metallic systems.
  • bandgap (Optional[float]) – Automatically adjust the band gap to this value. Cannot be used in conjunction with the scissor option. Has no effect for metallic systems.
  • symprec (float) – The symmetry tolerance used when determining the symmetry inequivalent k-points on which to interpolate.
  • nworkers (int) – The number of processors used to perform the interpolation. If set to -1, the number of workers will be set to the number of CPU cores.
Return type:

AmsetData
Returns:

The electronic structure (including energies, velocities, density of states and k-point information) as an AmsetData object.

get_energies(kpoints, energy_cutoff=None, scissor=None, bandgap=None, return_velocity=False, return_effective_mass=False, return_projections=False, return_vel_outer_prod=True, coords_are_cartesian=False, atomic_units=False, skip_coefficients=None)¶

Gets the interpolated energies for multiple k-points in a band.

Note, the accuracy of the interpolation is dependant on the interpolate_factor used to initialize the Interpolater.

Parameters:
  • kpoints (Union[ndarray, List[~T]]) – The k-point coordinates.
  • energy_cutoff (Optional[float]) – The energy cut-off to determine which bands are included in the interpolation. If the energy of a band falls within the cut-off at any k-point it will be included. For metals the range is defined as the Fermi level ± energy_cutoff. For gapped materials, the energy range is from the VBM - energy_cutoff to the CBM + energy_cutoff.
  • scissor (Optional[float]) – The amount by which the band gap is scissored. Cannot be used in conjunction with the bandgap option. Has no effect for metallic systems.
  • bandgap (Optional[float]) – Automatically adjust the band gap to this value. Cannot be used in conjunction with the scissor option. Has no effect for metallic systems.
  • return_velocity (bool) – Whether to return the band velocities.
  • return_effective_mass (bool) – Whether to return the band effective masses.
  • return_projections (bool) – Whether to return the interpolated projections.
  • return_vel_outer_prod (bool) – Whether to return the outer product of velocity, as used by BoltzTraP2 to calculate transport properties.
  • coords_are_cartesian (bool) – Whether the kpoints are in cartesian or fractional coordinates.
  • atomic_units (bool) – Return the energies, velocities, and effective_massses in atomic units. If False, energies will be in eV, velocities in cm/s, and effective masses in units of electron rest mass, m0.
Return type:

Union[Dict[Spin, ndarray], Tuple[Dict[Spin, ndarray], …]]
Returns:

The band energies as dictionary of:

{spin: energies}

If return_velocity, return_effective_mass or return_projections a tuple is returned, formatted as:

(energies, Optional[velocities], Optional[effective_masses],
 Optional[projections])

The velocities and effective masses are given as the 1x3 trace and full 3x3 tensor, respectively (along cartesian directions). The projections are summed for each orbital type (s, p, d) across all atoms, and are given as:

{spin: {orbital: projections}}
